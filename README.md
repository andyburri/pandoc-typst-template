# Pandoc Typst template

This template is based on the [Eisvogel template](https://github.com/Wandmalfarbe/pandoc-latex-template).

## Custom Template Variables

- `titlepage` (defaults to `false`)
  turns on the titlepage when `true`
- `titlepage-rule` (defaults to `2.5pt + black`)
  set the title page rule style
- `titlepage-bg` (defaults to `#fff`)
  sets the background color for the titlepage
- `titlepage-fg` (defaults to `#000`)
  sets the foreground color for the titlepage
- `toc` (defaults to `false`)
  inserts table of contents
- `toc-depth` (defaults to `none`)
  sets the depth for the toc
- `lot` (defaults to `false`)
  inserts list of tables
- `lof` (defaults to `false`)
  inserts list of figures
- `dozent`, `dozentin` (defaults to `none`)
  adds a Dozent line to the title page
